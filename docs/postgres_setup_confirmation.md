Download and install the latest version of PostgreSQL (version 17). Remember the password and username `postgres` and the port (which is 5432).
Install a graphical interface tool. I prefer something lightweight and with support for many database types, DBeaver is a very good choice (version 26.0.2).
Create an empty test database (platform_db_dev).
