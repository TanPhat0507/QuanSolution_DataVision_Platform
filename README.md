## 🎯 Overview
The core objective of this repository and role is to build a robust data foundation. The main responsibility is to make data **structured, reliable, queryable, and dashboard-ready** to drive accurate analytics and business intelligence.

## Key Ownership & Responsibilities
As the owner of database quality and analytics, the scope of work includes:

- **PostgreSQL Schema:** Design, deploy, and maintain logical and physical database schemas.
- **Database Tables:** End-to-end management of core data storage structures.
- **pgvector Setup:** Support the integration and configuration of vector data for advanced search capabilities.
- **Data Quality Checks:** Implement constraints, validations, and automated checks to ensure data integrity.
- **Analytics Views:** Build optimized relational views for fast analytical processing.
- **Power BI-Ready SQL Views:** Design tailored, pre-aggregated data views for seamless Power BI integration.
- **Dashboard Metric Queries:** Write and optimize complex SQL queries to calculate core business metrics.

## Tools & Technologies
- **PostgreSQL:** Core Database Management System (DBMS).
- **pgvector:** Open-source vector similarity search extension for PostgreSQL.
- **DBeaver:** Primary Database Management GUI for administration and querying.
- **SQL:** Core language for data definition, manipulation, and querying.
